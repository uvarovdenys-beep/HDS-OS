# HDS Protocol System — Архітектура розмірів

> Єдина система задач. Адаптивні протоколи. Психологічна діагностика.

---

## Філософія

```
ЗАДАЧА = одна для всіх моделей (єдиний YAML формат)
ПРОТОКОЛ = обгортка, яка адаптує задачу під когнітивний рівень моделі
ДІАГНОСТИКА = 6 тестів, які за 3 хвилини визначають розмір протоколу
```

Аналогія з одягом:
- Тіло (задача) — одне
- Розмір одягу (протокол) — підбирається під фігуру (модель)
- Примірочна (діагностика) — визначає розмір

## 4 Розміри протоколів

| Розмір | Клас моделі | Когнітивний стиль | Автономність |

## Єдиний формат задачі (YAML)

```yaml
task_id: TASK-001
goal: "Додати валідацію email у форму реєстрації"
priority: high
complexity: 6  # 1-10

target:
  file: "src/auth/register.py"
  line: 142
  function: "validate_input"

context: |
  Зараз форма приймає будь-який текст як email.
  Потрібна базова перевірка формату.

solution_hint: |
  Використай regex: r'^[\w\.-]+@[\w\.-]+\.\w+$'

test: |
  python3 -c "
  from src.auth.register import validate_input
  assert validate_input('test@mail.com') == True
  assert validate_input('not-email') == False
  print('PASSED')
  "

decomposition:  # тільки для XL/L
  - name: "Додати import re"
    target: "src/auth/register.py"
    line: 1
  - name: "Створити функцію validate_email()"
    target: "src/auth/register.py"
    line: 140
  - name: "Інтегрувати в validate_input()"
    target: "src/auth/register.py"
    line: 142
```

**Кожна модель бачить ту ж задачу, але через свою лінзу (протокол):**

- **XL** бачить: goal + context + decomposition → планує DVP
- **L** бачить: goal + target + solution_hint → робить за кроками
- **M** бачить: target + solution_hint + test → конкретна зміна
- **S** бачить: file + line + готовий код → copy-paste + test

## Діагностика (6 психологічних тестів)

Файл: `ai-mind/protocols/diagnostic.yaml`

Кожен тест оцінює одну когнітивну здатність.
Модель проходить всі 6 → набирає 0-18 балів → отримує розмір.

| Бали | Розмір |
|------|--------|
| 15-18 | XL |
| 10-14 | L |
| 6-9 | M |
| 0-5 | S |

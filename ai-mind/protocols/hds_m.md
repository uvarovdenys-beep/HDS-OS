# HDS M — Протокол для виконавців

> Розмір: M | Діагностичний бал: 6-9
> Для: Qwen 9B-14B, Llama 13B, Gemma 9B, Mistral (без 7B)

---

## BOOT (2 кроки)

```
1. Прочитай цей файл (правила + ліміти)
2. Прочитай свою задачу з ai-mind/tasks/active/
```

---

## МОЖЛИВОСТІ

- Структуроване виконання (конкретні інструкції)
- Зміна існуючих файлів
- Прості нові функції (до 50 рядків)
- Запуск тестів

---

## ПРОТОКОЛ: CHECK-DO-TEST

```
CHECK → Прочитай файл який будеш змінювати
DO    → Внеси ТІЛЬКИ ту зміну що в задачі
TEST  → Запусти тест (скопіюй з задачі)
```

---

## ПРАВИЛА (5)

M-01 ONE_TASK — одна задача, один файл
M-02 READ_FIRST — прочитай файл перед зміною
M-03 ONLY_ASKED — роби тільки те що просили
M-04 MAX_200_LINES — не більше 200 рядків
M-05 STOP_IF_CONFUSED — не розумієш → зупинись

---

## ЛІМІТИ

| AIVC | ❌ Заборонено |
| Нові модулі | ⚠️ Тільки прості (<50 рядків) |

---

## ФОРМАТ ЗАДАЧІ (як бачить M)

Задача максимально конкретна:
- File + Line → точне місце зміни
- Solution_hint → підказка рішення
- Test → готовий тест для копіювання

M НЕ бачить:
- decomposition (занадто складно)
- context (може заплутати)
